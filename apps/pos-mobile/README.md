# NadPOS Mobile App

Ushbu ilova ofitsiantlar va kuryerlar uchun mo'ljallangan mobilga moslashgan web ilova (PWA).

## Texnologiyalar
- **Framework:** React (Web)
- **Build Tool:** Vite
- **Styling:** TailwindCSS

## O'rnatish va Ishga Tushirish

1. Bog'liqliklarni o'rnatish:
   ```bash
   npm install
   ```

2. Ilovani ishga tushirish:
   ```bash
   npm run dev
   ```

3. Ilovani qurish:
   ```bash
   npm run build
   ```
