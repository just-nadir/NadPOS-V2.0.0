# NadPOS Admin Panel

Ushbu ilova restoran egalari va boshqaruvchilari uchun mo'ljallangan web-interfeys.

## Texnologiyalar
- **Framework:** React
- **Build Tool:** Vite
- **Styling:** TailwindCSS
- **Icons:** Lucide React

## O'rnatish va Ishga Tushirish

1. Bog'liqliklarni o'rnatish:
   ```bash
   npm install
   ```

2. Ilovani ishga tushirish (Development):
   ```bash
   npm run dev
   ```

3. Ilovani qurish (Production Build):
   ```bash
   npm run build
   ```
