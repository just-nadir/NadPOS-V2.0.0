# Product Context

## Purpose
NadPos-Restoran exists to streamline restaurant operations, replacing manual or outdated systems with a modern, integrated solution. It solves problems related to order accuracy, inventory management, and sales reporting.

## Target Audience
-   **Restaurant Staff (Waiters/Cashiers)**: Need a fast, intuitive interface for entering orders and processing payments.
-   **Restaurant Managers/Owners**: Need real-time insights into sales, inventory control, and employee performance.

## Key Features
-   **Table Management**: Visual representation of restaurant layout and table status.
-   **Order Taking**: Quick menu navigation and order modification.
-   **Offline Capability**: The desktop POS should function even with intermittent internet access, syncing data when connectivity is restored.
-   **Centralized Management**: Cloud admin panel for configuration and reporting.

## User Experience Goals
-   **Speed**: Operations must be swift to keep up with peak restaurant hours.
-   **Reliability**: The system must be stable and data integrity is paramount.
-   **Ease of Use**: Minimal training required for new staff.
